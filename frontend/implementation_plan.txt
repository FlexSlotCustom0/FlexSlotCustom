# FlexSlot - Live Template Editor Implementation Plan

This plan aims to transform the 6 static business templates into a dynamic "Shop Builder" experience. Shop owners will be able to click a "Customize" button and edit their brand name, colors, images, banner, staff, and services in real-time.

## User Review Required

> [!IMPORTANT]
> The current templates use hardcoded constants. This refactor will move all shop data into a React Context. While this enables the "Live Editor," it changes how data is initialized. I will ensure each template still feels unique but shares a common editing engine.

---

## Proposed Changes

1. Unified Shop Context (TemplateProvider.tsx)
2. Interactive Editor Sidebar (TemplateEditor.tsx)
3. Refactoring 6 templates to use shared state.
4. Floating "Edit" button for all template pages.
